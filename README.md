# 🛒 Product Cart - Angular 21

> A modern product browsing and cart application demonstrating Angular 21 standalone components, component communication, and the new control flow syntax.

[![Angular](https://img.shields.io/badge/Angular-21-red?style=flat&logo=angular)](https://angular.io/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.2-blue?style=flat&logo=typescript)](https://www.typescriptlang.org/)
[![Standalone](https://img.shields.io/badge/Components-Standalone-green?style=flat)](https://angular.io/guide/standalone-components)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## 🎯 Features

-  **Standalone Components** - Modern Angular 21 architecture
-  **Component Communication** - @Input and @Output decorators
-  **New Control Flow** - @if, @for, @else syntax
-  **Real-time Search** - Filter products as you type
-  **Cart Management** - Add/remove products with visual feedback
-  **Responsive Design** - Mobile-first CSS Grid layout
-  **Type Safety** - Full TypeScript implementation

## 🚀 Quick Start

\`\`\`bash
npm install
ng serve
\`\`\`

Open [http://localhost:4200](http://localhost:4200) in your browser.
